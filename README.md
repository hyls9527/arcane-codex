# Arcane Codex

> 你的照片，你的记忆，你的知识库

我拍了很多照片。硬盘里躺着几万张，手机相册里还有几万张。找一张特定照片的时候，我就像在垃圾堆里翻宝贝——明明知道它在那里，就是找不到。

Arcane Codex 是我为自己写的工具。它用 AI 理解照片内容，用语义搜索找到你想要的那一张，全部运行在本地，你的数据永远属于你。

## 它能做什么

**🧠 看懂你的照片**

扔进去一张图，它能告诉你：这是猫还是狗？室内还是室外？风景照还是自拍？不只是打标签，而是真正理解画面内容。

支持 6 种 AI 后端：LM Studio、Ollama、Hermes、智谱、OpenAI、OpenRouter。本地的、云端的，随你选。

**🔍 像说话一样搜索**

不用记文件名，不用翻文件夹。输入"去年夏天的海边"、"那只白猫"、"开会时的白板"，它懂你在找什么。

**🗂️ 自动整理**

重复的照片？自动标出来。相似的图片？归到一起。几万个文件？照样秒开。

## 为什么写这个

市面上的照片管理工具要么太简单（系统自带的），要么太复杂（企业级 DAM），要么把照片传到别人的服务器上（各种云相册）。

我想要的是：
- 完全本地运行，隐私无忧
- AI 理解内容，不是简单的文件名搜索
- 轻量快速，不臃肿
- 开源，代码透明

所以我自己写了一个。

## 技术栈

- **前端**: React + TypeScript + Tailwind CSS
- **后端**: Rust + Tauri 2.0
- **数据库**: SQLite
- **AI**: 支持 OpenAI 兼容 API 的任何服务

## 快速开始

```bash
# 克隆
git clone https://github.com/hyls9527/arcane-codex.git
cd arcane-codex

# 安装依赖
npm install

# 启动开发模式
npm run tauri dev
```

## 使用

1. **配置 AI**: 设置 → AI 配置，填入你的 API Key 或启动本地服务
2. **导入照片**: 拖拽文件夹到窗口，或点击导入按钮
3. **等待分析**: AI 会自动为每张照片生成标签和描述
4. **开始搜索**: 在搜索框里输入任何描述，比如"那只橘猫"
